# TikTok Auto Upload (Краткая инструкция)

Этот бот автоматически загружает видео на TikTok, используя аккаунты из `cookies/` и видео из `videos/`.

## Быстрый старт

1. **Подготовка окружения**:
   - Убедитесь, что у вас заполнен файл `.env` своими данными (прокси, настройки).
   - Файлы cookies положите в папку `cookies/` (поддерживаются JSON и Netscape).
   - Видео положите в папку `videos/`.

2. **Установка зависимостей**:
   ```bash
   pip install -r requirements.txt
   playwright install firefox
   ```

3. **Запуск**:
   ```bash
   python main.py
   ```

## Как это работает
- Скрипт берет **необработанные** файлы из `cookies/` (без префикса `valid_` или `invalid_`).
- Для каждого аккаунта берет первое видео из `videos/`.
- Меняет IP через прокси (если настроено).
- Загружает видео на TikTok.
- После работы переименовывает файл куки в `valid_...` или `invalid_...`.

## Важно
- Если скрипт пишет "Не найдено файлов с куками", переименуйте ваши файлы в папке `cookies`, убрав префиксы `valid_`/`invalid_`, чтобы запустить их снова.
