# Заметки: get_tiktok_views.py

## Текущая реализация
- Использует aiohttp для HTTP-запросов (без браузера)
- Парсит JSON из HTML страницы TikTok
- Работает быстро, не требует Playwright

## Лимиты TikTok

### Без прокси (текущий режим)
- Единичные запросы — работает без проблем
- Массовые запросы (десятки-сотни подряд) — возможны проблемы:
  - Капча или пустые ответы
  - Временная блокировка IP (5-15 минут)
  - Требование JavaScript-рендеринга

### Рекомендации для batch-режима
1. Добавить задержку между запросами (1-3 секунды)
2. Ротировать User-Agent
3. При ошибках — использовать прокси или Playwright как fallback
4. Опционально: использовать прокси из config.py
